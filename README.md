## Skills/Expertise
* [Algorithmic Art](skills/algorithmic-art/SKILL.md) – Generative art with p5.js, seeded randomness, and interactive parameter exploration.
* [Brand Guidelines](skills/brand-guidelines/SKILL.md) – Apply Anthropic brand colors and typography to artifacts.
* [Canvas Design](skills/canvas-design/SKILL.md) – Create design philosophies and express them as polished visual art (PNG/PDF).
* [Doc Coauthoring](skills/doc-coauthoring/SKILL.md) – Structured workflow for drafting and refining docs, specs, and proposals.
* [DOCX](skills/docx/SKILL.md) – Create, edit, and analyze Word documents with tracked changes and formatting fidelity.
* [Frontend Design](skills/frontend-design/SKILL.md) – Build distinctive, production-grade web interfaces without generic AI aesthetics.
* [Internal Comms](skills/internal-comms/SKILL.md) – Patterns and examples for effective company communications.
* [MCP Builder](skills/mcp-builder/SKILL.md) – Guidance for designing and implementing MCP servers and tools.
* [PDF](skills/pdf/SKILL.md) – Toolkit for extracting, generating, and manipulating PDFs, including form handling.
* [PPTX](skills/pptx/SKILL.md) – Presentation creation, editing, and analysis workflows for PowerPoint files.
* [Skill Creator](skills/skill-creator/SKILL.md) – Instructions for authoring new skills with concise, effective workflows.
* [Slack GIF Creator](skills/slack-gif-creator/SKILL.md) – Build animated GIFs optimized for Slack dimensions and constraints.
* [Template Skill](skills/template/SKILL.md) – Starter template for creating new skills.
* [Theme Factory](skills/theme-factory/SKILL.md) – Apply curated color/font themes or craft new ones for artifacts.
* [Web Artifacts Builder](skills/web-artifacts-builder/SKILL.md) – React/Tailwind/shadcn toolkit for bundling complex claude.ai HTML artifacts.
* [Webapp Testing](skills/webapp-testing/SKILL.md) – Playwright-based workflows for testing local web applications.
* [XLSX](skills/xlsx/SKILL.md) – Create and edit spreadsheets with formulas, formatting, and recalculation support.
